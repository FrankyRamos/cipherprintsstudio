<h1>Privacy Policy &mdash; CipherPrintsStudio</h1>
<p class="meta">Effective date: 2026-05-12 &middot; Last updated: 2026-05-12</p>

<p>This privacy policy explains how <strong>CipherPrintsStudio</strong> (the "Service", operated by an individual seller) handles information in connection with our personal-use Pinterest API integration ("CipherPrintsStudio Auto Pinner") and our Etsy print-on-demand shop at <a href="https://www.etsy.com/shop/CipherPrintsStudio">etsy.com/shop/CipherPrintsStudio</a>.</p>

<h2>1. Scope</h2>
<p>The Auto Pinner is a personal automation tool. It posts pins of CipherPrintsStudio's own Etsy listings to a single Pinterest board owned by CipherPrintsStudio. It does not interact with other Pinterest users, does not collect data from any third parties, and is not offered as a service to anyone else.</p>

<h2>2. Information we do NOT collect</h2>
<ul>
  <li>We do not collect personal data from Pinterest users other than the operator of the CipherPrintsStudio account.</li>
  <li>We do not use Google Analytics, Mouseflow, or any other tracking or analytics tool.</li>
  <li>We do not show ads.</li>
  <li>We do not send marketing emails.</li>
  <li>We do not collect location, contacts, or camera data.</li>
  <li>We do not knowingly collect any information from children under 13.</li>
</ul>

<h2>3. Information we use (operator only)</h2>
<p>To operate the Auto Pinner, the following data is stored locally on the operator's machine:</p>
<ul>
  <li>A Pinterest API access token, used to authenticate API calls to the operator's own Pinterest account.</li>
  <li>The operator's Pinterest board ID for the "Etsy Shop" board.</li>
  <li>Public Etsy listing data (title, description, image, and listing URL) of CipherPrintsStudio's own Etsy listings.</li>
</ul>
<p>This data is never shared with third parties.</p>

<h2>4. How pins are created</h2>
<p>The Auto Pinner pulls public listing data from CipherPrintsStudio's own Etsy shop and creates Pinterest pins on CipherPrintsStudio's own "Etsy Shop" board, with each pin linking back to the corresponding Etsy listing page. No content is posted on behalf of any third party.</p>

<h2>5. Cookies</h2>
<p>This privacy policy page does not use cookies or any client-side tracking.</p>

<h2>6. Data retention</h2>
<p>Locally stored credentials (Pinterest access token, board ID) persist until the operator deletes them. They are not transmitted to any party other than Pinterest's own API endpoints.</p>

<h2>7. Children</h2>
<p>This service is not directed to children under 13 and does not knowingly collect information from them.</p>

<h2>8. Changes to this policy</h2>
<p>If this policy changes, the "Last updated" date at the top of this page will be revised.</p>
